== To setup C# script template: ==
1. Move 81-C# Script-NewBehaviourScript.cs.txt to C:\Program Files\Unity\Hub\Editor\youreditorversion\Editor\Data\Resources\ScriptTemplates

== To setup Unity Editor Layout: ==
1. Move Best.wlt to C:\Users\yourusername\AppData\Roaming\Unity\Editor-5.x\Preferences\Layouts\default

== To setup Visual Studio Settings: ==
1. Open Visual Studio.
2. Go to Tools > Import and Export Settings Wizard.
3. Select Import selected environment settings.
4. Follow the on screen instructions.
